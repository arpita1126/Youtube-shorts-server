# Backend - UltraLite Shorts

This folder will contain a small Node.js / Express server that mimics the Next.js API routes and provides endpoints:

- GET /feed
- GET /network-profile
- POST /events/log

Quick start:

1. cd backend
2. pnpm install
3. cp .env.example .env and edit values if needed
4. pnpm dev

.env variables (example)

PORT=4000
LOG_RETENTION=1000
